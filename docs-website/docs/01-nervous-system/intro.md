# Module 1: The Nervous System (ROS 2)

**Foundation of Robot Communication & Control**

## Learning Objectives

- Understand ROS 2 architecture (nodes, topics, services)
- Write a simple ROS 2 node in Python
- Deploy on Unitree Go2/G1 and Jetson Orin

## Prerequisites

- Basic Linux/Ubuntu knowledge
- Python 3.8+
- GPU: RTX 4070 Ti or Jetson Orin

## Key Topics

1. **ROS 2 Fundamentals**: Installation, workspaces, packages
2. **Pub/Sub Model**: Topics, subscribers, publishers
3. **Services & Actions**: Request-response patterns
4. **Sensor Integration**: Camera, LiDAR, IMU
5. **Real Robot Deployment**: Go2/G1 robot setup
