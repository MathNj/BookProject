# Module 2: Digital Twin (Simulation)

**Building Virtual Replicas of Physical Systems**

This module introduces digital twins—exact virtual replicas of physical robots that enable safe testing, training, and optimization before deploying to hardware.

## Learning Objectives

- Design and simulate robot dynamics in Isaac Sim
- Train policies in simulation and transfer to real robots
- Validate sensor integration and control loops

## Prerequisites

- Completion of Module 1: The Nervous System
- RTX 4070 Ti or Jetson Orin
- 30GB+ SSD space for simulation assets

## Key Topics

1. **Simulation Fundamentals**: Physics engines, rendering
2. **Isaac Sim Environment**: Robot urdf, scene setup
3. **Sensor Simulation**: Camera, LiDAR, IMU in simulation
4. **Control Loop Debugging**: Safe testing before real hardware
5. **Sim-to-Real Transfer**: Policies learned in simulation → deployed on Go2/G1
